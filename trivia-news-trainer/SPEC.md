# Trivia News Trainer - Specification

## Project Overview
- **Name:** Trivia News Trainer
- **Purpose:** Training app for trivia teams with top 5 news stories per category
- **Target Users:** Trivia team members (competitive/NAQT style)

## Data Structure
Each category has 5 stories with:
- **Title:** Short headline description
- **Details:** Expandable dropdown with full summary

## Categories (Pilot)
1. Politics
2. World  
3. Business

## UI/UX
- Clean, modern interface
- Category tabs navigation
- Expandable story cards with dropdowns
- Mobile-responsive

## Tech Stack
- Next.js 14 (App Router)
- Single page, no backend needed
- Static JSON data
